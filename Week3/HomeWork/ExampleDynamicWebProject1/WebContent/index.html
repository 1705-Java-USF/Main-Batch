Servlets are interacted with via form submissions
Inside the form start tag, you can optionally set a method property
But it is the action tag, that will direct us to a servlet to handle
Note: action should equal to the servlet URL pattern, as indicated in the web.xml

POST AND GET

Utilize the 'method' attribute for a form to indicate which http method to use
If no method is specified, GET is use by default!
