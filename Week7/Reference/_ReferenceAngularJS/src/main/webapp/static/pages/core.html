Core Directives

ng-app

Simply serves to designate an html file (or html file part) as and angular application. Allowing for use of it's functions


ng-init

Designate a default value for some model date for use at page load. In this example I have set the default model for "init1" to be ""Bobbert".

init1 default:
{{init1}}

ng-model

When used with user input, we can bind the input to variable names as used by angular

+ = {{ 0 + num1 + num2}}
- = {{ 0 + (num3 - num4)}}
* = {{ 0 + (num5 * num6)}}
/ = {{ 0 + (num7 / num8)}}

ng-repeat

Acts as a for:each loop. Will repeat all elements within for each item in a collection.
